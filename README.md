# RaiRuby

An RPC wrapper for RaiBlocks written in Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rai_rpc'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rai_rpc
